# Employee Management System

A Spring Boot–based REST API application to manage employee data efficiently. This project demonstrates backend development using Java, Spring Boot, and database integration.

---

## 📌 Features

- Add Employee
- View All Employees
- Get Employee by ID
- Update Employee Details
- Delete Employee

---

## 🛠 Tech Stack

- Java
- Spring Boot
- Spring Data JPA
- H2 Database
- Maven
- Postman (for API testing)

---

## 🚀 API Endpoints

| Method | Endpoint | Description |
|--------|---------|------------|
| POST   | /employees        | Add new employee |
| GET    | /employees        | Get all employees |
| GET    | /employees/{id}   | Get employee by ID |
| PUT    | /employees/{id}   | Update employee |
| DELETE | /employees/{id}   | Delete employee |
